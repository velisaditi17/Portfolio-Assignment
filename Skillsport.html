HTML (HyperText Markup Language) is the standard language used to create and structure web pages.
It tells a web browser how to display text, images, links, tables, forms, and other content on a webpage.

1. Easy to learn and use

2. Uses tags to define elements

3. Forms the structure of a webpage

4. Works together with

CSS (Cascading Style Sheets) is a stylesheet language used to control the appearance and layout of web pages.
It is used along with HTML to make websites attractive and responsive.

1. To add colors, fonts, and backgrounds.

2. To control the layout of web pages.

3. To make websites responsive on different devices.

4. To separate content (HTML) from presentation (CSS).

JavaScript (JS) is a high-level, interpreted programming language used to make web pages interactive and dynamic.
It is one of the core technologies of web development, along with HTML and CSS.

1. To add interactivity to web pages.

2. To validate forms before submission.

3. Lightweight and easy to learn.

4. Runs directly in web browsers.

Bootstrap is a free and open-source front-end framework used for developing responsive and mobile-first websites.

1. Faster web development.

2. Consistent design across browsers.

3. Easy customization.

4. Reduces the amount of CSS code needed.

It is widely used for developing desktop applications, web applications, mobile applications, and enterprise software.

1. To develop web applications.

2. To create Android applications.

3. To build enterprise software.

4. To create backend services and APIs.
